# Forward KL (FKL) Distillation with Dynamic Masking

## Goal
The objective is to distill knowledge from a "teacher" model into a smaller "student" model (Qwen3-4B). Specifically, we want the student to learn to produce high-quality responses ($y^*$) using only the initial prompt ($x$), effectively internalizing improvements that the teacher made when it had access to extra context (original responses $y$ and follow-up feedback $o$).

## Mathematical Formulation

### 1. The Distillation Objective
We minimize the Forward Kullback-Leibler (KL) divergence between the teacher's distribution $q$ and the student's distribution $p_\theta$:

\[ \min_\theta \text{KL}(q || p_\theta) = \min_\theta \mathbb{E}_{y^* \sim q} [ -\log p_\theta(y^* | x) ] \]

Where:
*   $x$: Initial prompt.
*   $y^*$: Target response generated by the teacher (conditioned on $x, y, o$).
*   $p_\theta(y | x)$: Student model likelihood.

### 2. Dynamic Confidence Masking
To handle "off-policy" samples where the student model is extremely uncertain, we apply a dynamic mask $m_i$ to each token $i$ in the target sequence $y^*$:

\[ m_i = \begin{cases} 1 & \text{if } p_\theta(y^*_i | x, y^*_{<i}) \ge \tau \\ 0 & \text{otherwise} \end{cases} \]

The hyperparameter $\tau$ (`mask_tau`) determines the strictness of the filter. If the student's current probability for the teacher's token is below $\tau$, that token is ignored during the gradient update.

### 3. The Masked Loss Function
The final loss for a sequence is the negative log-likelihood of the unmasked tokens, normalized by the number of active tokens:

\[ \mathcal{L}_{masked}(\theta) = -\frac{\sum_{i=1}^{T} m_i \cdot \log p_\theta(y^*_i | x, y^*_{<i})}{\sum_{i=1}^{T} m_i} \]

## Implementation Details
*   **Model:** Qwen3-4B
*   **Infrastructure:** FSDP (Fully Sharded Data Parallel) ZeRO-3 sharding.
*   **Optimization:** 2x GPUs, supporting both 8-bit AdamW (for memory efficiency) and 32-bit AdamW (for precision).
*   **Masking:** A threshold of $\tau = 0.001$ typically filters out ~90% of tokens in early training, while $\tau = 0.0001$ or $\tau = 0$ (Standard SFT) provides more dense signal.

## Current Experiments
We are currently running 6 variations to compare the impact of the masking threshold and optimizer precision:
1.  **Baseline SFT:** $\tau = 0$, Precise AdamW.
2.  **Relaxed FKL:** $\tau = 0.0001$, Precise AdamW.
3.  **Mid-range FKL:** $\tau = 0.0005$, Precise AdamW.
4.  **Aggressive LR:** $\tau = 0.0001$, LR $= 5 \times 10^{-6}$.
5.  **Very Aggressive LR:** $\tau = 0.0001$, LR $= 10^{-5}$.
6.  **Optimizer Comparison:** $\tau = 0.0001$, 8-bit AdamW.
